Trello Power-Up

Checklist Companion

Use this page as the iframe connector URL for a custom Trello Power-Up that shows checklist headers and checklist items directly on the front of Trello cards.

  1. Create a custom Power-Up in https://trello.com/apps/admin.
  2. Set this file's hosted URL as the iframe Connector URL.
  3. Generate an API key for the Power-Up, add https://dxnielll.github.io to the allowed origins, and paste the key into config.js.
  4. Enable card-badges and show-settings.
  5. Add the Power-Up to a board in the same Workspace, open Power-Up settings, and connect Trello access once.

If you are opening this page directly in a browser, it is working as expected. Trello will load it silently as the connector iframe.
