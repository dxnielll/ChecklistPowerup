Trello Power-Up

Checklist Companion

Use this page as the iframe connector URL for a custom Trello Power-Up that shows checklist progress and checklist items directly on the front of Trello cards.

  1. Create a custom Power-Up in https://trello.com/apps/admin.
  2. Set this file's hosted URL as the iframe Connector URL.
  3. Enable card-badges, card-back-section, and show-settings.
  4. Add the Power-Up to a board in the same Workspace and open Power-Up settings to configure the front-of-card display.

If you are opening this page directly in a browser, it is working as expected. Trello will load it silently as the connector iframe.
